Projeto Monalisa

O Projeto Monalisa é uma iniciativa de codificação criativa que se propõe a dar vida digital a uma caricatura. Seu conceito central é uma representação artística colorida que simula o famoso efeito do olhar da Mona Lisa, onde os olhos do personagem parecem seguir o movimento do espectador.

A estrutura para o desenvolvimento deste projeto é o p5.js, uma biblioteca JavaScript amplamente utilizada no campo da programação.

A implementação da movimentação dos olhos envolve o uso de variáveis como mouseX e mouseY (que capturam as coordenadas do mouse) e funções trigonométricas, como atan2(), para calcular os ângulos necessários e girar as pupilas dentro das órbitas oculares da caricatura.

Projeto Jogo Ping Pong

O Projeto Jogo Ping Pong é uma implementação digital do clássico jogo de tênis de mesa, desenvolvido para dois jogadores. Sua essência reside na recriação fiel das mecânicas fundamentais do Ping Pong: duas raquetes controladas pelos jogadores, uma bola em constante movimento e um sistema de pontuação para registrar os sucessos.

A estrutura para o desenvolvimento deste projeto é o p5.js, uma biblioteca JavaScript amplamente utilizada no campo da programação.

O p5.js é particularmente adequado para jogos bidimensionais como o Ping Pong. Ele simplifica tarefas essenciais como desenhar as raquetes (retângulos) e a bola (círculo), animar seus movimentos suaves na tela, detectar colisões precisas entre a bola e as raquetes ou as bordas do "campo", e processar as entradas do teclado para que os jogadores possam controlar suas respectivas raquetes. Toda essa estrutura de código é orquestrada por funções como setup(), que inicializa o jogo, e draw(), que atua como o loop principal, atualizando constantemente o estado do jogo e renderizado os elementos visuais

Projeto Aluraflix

O Projeto AluraFlix é um site desenvolvido com a proposta de ser uma plataforma de streaming de vídeo. O projeto reside em seu público-alvo específico: crianças. O AluraFlix oferece um ambiente seguro e divertido para crianças, apresentando um vasto catálogo de vídeos de desenhos animados.

Todo o desenvolvimento do Projeto AluraFlix foi realizado GitHub, uma plataforma que permitiu a colaboração eficiente da equipe e o gerenciamento de versões do código. A utilização do GitHub garante a transparência do processo de desenvolvimento, a facilidade de manutenção e a possibilidade de futuras melhorias e expanções do projeto.

Com o AluraFlix, as crianças têm à disposição uma biblioteca de vídeos de desenhos animados que podem ser acessados de forma simples e intuitiva, proporcionando horas de entreterimento de qualidade.
